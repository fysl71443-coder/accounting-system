## 🚀 كيفية التشغيل - How to Run

### 💻 **التطوير المحلي**
```bash
# 1. تشغيل سريع
python run_local.py

# 2. أو تشغيل يدوي
pip install -r requirements.txt
python app.py
```

### ☁️ **النشر على Render**
```bash
# 1. إعداد النشر
python setup_deployment.py

# 2. رفع إلى GitHub
git add .
git commit -m "Deploy to Render"
git push origin main

# 3. اتبع دليل النشر
# راجع DEPLOYMENT_GUIDE.md
```
